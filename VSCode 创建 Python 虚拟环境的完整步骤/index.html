    将 VSCode 创建 Python 虚拟环境的完整步骤

    在 VSCode 中创建 Python 虚拟环境的完整步骤

    1. 安装必要的工具

    在开始之前，请确保以下工具已安装： - Python：可以从 Python 官方网站 下载。 - Visual Studio Code (VSCode)：可以从 VSCode 官方网站 下载。

    2. 打开 VSCode

    启动 Visual Studio Code，并确保你已经安装了 Python 插件。如果没有安装，可以通过以下步骤添加： 1. 点击左侧菜单中的扩展图标（四个方块组成的图标）。 2. 搜索 Python 并安装 Microsoft 提供的官方插件。

    3. 创建新文件夹和项目

    1. 在 VSCode 中，按 Ctrl + Shift + N 打开终端。
    2. 输入以下命令创建一个新的文件夹： bash mkdir my_project cd my_project
    3. 初始化一个新的 Python 项目（可选）： bash python -m venv venv

    4. 创建虚拟环境

    1. 在终端中输入以下命令创建虚拟环境： bash python -m venv venv
    2. 激活虚拟环境：
    3. Windows: cmd .\venv\Scripts\activate
    4. macOS/Linux: bash source venv/bin/activate

    5. 配置 VSCode 的解释器

    1. 按 Ctrl + Shift + P 打开命令面板。
    2. 输入并选择 Python: Select Interpreter。
    3. 在弹出的列表中，选择你刚刚创建的虚拟环境（路径类似于 venv）。

    6. 安装依赖包

    1. 在激活的虚拟环境中，使用 pip 安装所需的依赖包。例如： bash pip install requests
    2. 你可以将依赖包列表保存到 requirements.txt 文件中： bash pip freeze > requirements.txt

    7. 使用虚拟环境

    现在你可以在 VSCode 中使用虚拟环境进行开发了！所有安装的包都会被限制在虚拟环境中，不会影响全局 Python 环境。

    8. 停用虚拟环境

    当你完成工作后，可以停用虚拟环境：

    deactivate

    9. 删除虚拟环境（可选）

    如果你不再需要这个虚拟环境，可以将其删除：

    rm -rf venv

    希望以上步骤对你有帮助！如果有任何问题，请随时提问。
